Política de Privacidade e LGPD

Este documento descreve as práticas de coleta, uso, armazenamento e compartilhamento de dados pessoais dos usuários do nosso sistema, conforme a Lei Geral de Proteção de Dados - LGPD.

1. Coleta de Dados

Coletamos informações como nome, data de nascimento, telefone, e-mail e demais dados que você fornecer ao utilizar o sistema. Essas informações são essenciais para o gerenciamento dos membros, batismos e relatórios.

2. Uso dos Dados

Os dados coletados são utilizados para:

  • Gerenciamento interno de membros e batismos;
  • Envio de notificações e lembretes;
  • Elaboração de relatórios e análises estatísticas;
  • Outros fins operacionais relacionados ao funcionamento do sistema.

3. Armazenamento e Segurança

Os dados são armazenados de forma segura em servidores e base de dados protegidos. Implementamos medidas de segurança, como criptografia, para prevenir acessos não autorizados.

4. Direitos dos Usuários

Você tem direito a:

  • Acessar as informações que temos sobre você;
  • Corrigir informações incorretas;
  • Solicitar a exclusão dos seus dados, quando aplicável;
  • Entre outros direitos previstos na LGPD.

5. Compartilhamento de Dados

Seus dados não serão compartilhados com terceiros sem o seu consentimento, exceto quando requerido por lei.

Consentimento

Ao utilizar este sistema, você concorda com os termos descritos nesta política de privacidade. Caso tenha dúvidas, entre em contato com nossa equipe de suporte.
